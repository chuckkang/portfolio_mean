Appointment App

Appointment application will allow a user to login and manage appointments.

Tech Notes

App coded in Angular 4, bootstrap, javascript and HTML. The app accesses a Mongo database

Funtional Notes

Only logged in user has ability to manage their own appointments.

Tech Notes

App coded in Angular 4, bootstrap, Javascript, HTML, CSS. Implements MongoDB. Angular uses a front-end service to make calls to an Node.js API. Node then returns a JSON object which is then stored in a service.
